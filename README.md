# SuperWindow
В приложении реализован следующий функционал:
  обработка сообщений;
  вывод текста в окно;
  отображение битового изображения;
  создание кастомного курсора и лого;
  создание меню;
  обработка горячих клавиш;
  реализация HANDLE_MSG;
  создание модального диалогового окна;
  использование стандартного диалогового окна
  "Open File";
  отображение текста подсказок в главном окне.
